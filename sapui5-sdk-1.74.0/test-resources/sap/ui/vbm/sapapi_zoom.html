SAP UI5 Visual Business: zoomToGeoPosition Test

The aim:

To test if on page load the GeoMap control is already zoomed in on a particular area; in this case, the area around Munich.


To test:

  1. Click on "Add 3 Spots" button. Check that 3 spots are added to the GeoMap control.
  2. Click the "Zoom Test" button. This will apply zoomToGeoPosition. The GeoMap should be zoomed into Waldorf.
  3. Click the "Automated Zoom Test" button.
  4. Click the "Automated Context Menu Test" button, and check that a Context menu appears.
  5. Click the "Switch Map" button, and check that the map switches.
